Print(), основы

Команда print служит для вывода информации в консоль, она может выводить числа, слова и символы.

?
1
2
3
#Не пугайтесь слов после решётки они комментируют код(подробнее далее)
print("Hello world!") #Выведет: Hello world!
print(123) #Выведет: 123

Текст необходимо писать в кавычках, а цифры можно и без них.

Переменные

Переменные в Python нужны для хранения информации, если слов, то с кавычками, а цифры можно хранить и без них.

?
1
2
3
x = "Приветствую тебя, мой юный падаван" #Неожиданно, но хранит текст
planet = "Tatuin"
bender = 1011000110101001

Print c переменными

Можно вывести текст с переменными, для этого его нужно обнести запятыми или добавить + после текста

?
1
2
3
4
5
planet = "Татуине"
print("Энакин живёт на ", planet)
#или
print("Энакин живёт на "+ planet)
#Вывод у обоих вариантов один: Энакин живёт на Татуине

Input(), основы

Метод input позволяет принимать для хранение в переменных информацию, отсортированную по типу, например: int для целых чисел

?
1
2
x = int(input() )#только числа, иначе ошибка
y = input() #Примет всё(Естественно в переменную y)!
